## 五、P3 拆分: 15-25 轮开发

### Phase 1: 基础设施补齐 (P3-1 ~ P3-3, 3 轮)
- P3-1: PostgreSQL+pgvector 迁库 (把 P2-1 的 SQLite 切到 PG+vector 扩展)
- P3-2: API 网关 + 服务注册 (新建 api-gateway, 8000 端口, JWT 路由)
- P3-3: 12 微服务 monorepo 拆分 (12 个服务独立启动)

### Phase 2: Agent 编排 (P3-4 ~ P3-6, 3 轮)
- P3-4: Agent 调度框架 (调度 Agent + 任务路由)
- P3-5: 15 Agent 实现 (每个 1 引擎 + API)
- P3-6: Agent 协作机制 (全自动/半自动/手动 3 模式)

### Phase 3: 算子库 (P3-7 ~ P3-11, 5 轮)
- P3-7: 32 清洗算子真实现
- P3-8: 20+ 标注算子
- P3-9: 15 评分算子 + 10 筛选算子
- P3-10: 12 导出算子 + 10 评测算子
- P3-11: 15 采集算子 (OpenMontage/Bernini/Prompt-Optimizer 整合)

### Phase 4: 工作流 (P3-12 ~ P3-14, 3 轮)
- P3-12: 工作流引擎 (DAG + Prefect 接入)
- P3-13: Vue Flow 可视化编辑器
- P3-14: 50 预置模板

### Phase 5: 前端重写 (P3-15 ~ P3-18, 4 轮)
- P3-15: Vue 3 + TS + Pinia + Naive UI 基础
- P3-16: 12 微服务对应的 12 个前端模块
- P3-17: 全功能 Agent 控制台
- P3-18: 工作流可视化 + 模板市场

### Phase 6: 部署 (P3-19 ~ P3-21, 3 轮)
- P3-19: K8s manifest + Helm chart
- P3-20: Prometheus + Grafana + Jaeger
- P3-21: 真实集群部署验证

**总计: 21 轮 (P3-1~P3-21) + 后续 1-4 轮运营 = 25 轮以内**
