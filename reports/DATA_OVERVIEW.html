全景总览

一个面向多模态大模型训练数据的工业级端到端平台。从原始素材采集 (collection) 到模型可消费的训练集 (COCO / YOLO / LLaVA / InternVL / WebDataset) 全链路打通, 47 个可复用能力 + 9 个核心引擎 + 14 张 ORM 表 + 2 个调度层 (APScheduler / TaskQueue) + 跨模态 RAG,跨进程持久,真上线 ready。

① 数据生产格式 ② 数据结构 ③ 9 阶段数据管线 ④ 8 大引擎 ⑤ 47 个能力 ⑥ RAG / 搜索 / Provider 栈 ⑦ 持久化层

① 数据生产格式

平台原生支持 8 种业务模态 + 解析层 7 种输入模态;生成侧注册 4 个 AI Provider;训练侧导出 9 种标准训练集格式。

8 个业务模态 (multimodal_v2.Modality)

image

→ video_engine (单帧提取)
输入: jpg / png / webp / gif / bmp
输出: image + mask
训练格式: COCO · YOLO · LLaVA · InternVL

video

→ video_engine
输入: mp4 / avi / mov / mkv / webm
输出: video + thumbnail + audio_track
训练格式: WebDataset (tar shard)

text

→ agent_router
输入: txt / md / json / jsonl
输出: text + token_ids
训练格式: JSONL (SFT/DPO)
requires_review=false

audio

→ audio_engine
输入: mp3 / wav / flac / m4a / ogg
输出: audio + transcript (Whisper ASR)
训练格式: JSONL (TTS/ASR)

multimodal

→ multimodal_router
输入: 任意组合
输出: image + text + audio
训练格式: LLaVA · InternVL

sketch

→ scene_exporter (3D 草图)
输入: 草图 + 描述
输出: sketch + metadata
导出: GLB · glTF · OBJ

drama

→ drama_engine (短剧)
输入: 剧本大纲
输出: script + storyboard + voiceover + final_cut
训练格式: InternVL · JSONL

picturebook

→ book_engine (绘本)
输入: 文字脚本
输出: pages + illustrations + characters
训练格式: Parquet · JSONL

C2PA Provenance 链路 (engines/c2pa_engine.py)

完整的 内容真实性 链路 — sign_asset / verify_signature / revoke / list_manifests / get_crl / cert_pem。 平台用 HMAC-SHA256 自签,自动 derive 证书 fingerprint,记录每个资产的来源、模型版本、操作者。
