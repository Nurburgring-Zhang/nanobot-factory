4. **剩余 `as any` 106 处**: 均为合法使用:
   - 跨节点 data lookup: `(n?.data as any) || {}` — 读其他节点的 data 时 xyflow 类型过宽
   - React Flow measure access: `(me as any)?.measured?.width` — 读运行时 layout 信息
   - Event handle: `(e as any).sourceHandle` — 读 drag/drop 事件
   - Enum cast: `e.target.value as any` — 缩窄 union 类型
   - 资源库记录: `(saved as any).duplicate` — 第三方 store 返回类型松散
5. **保留 IO CONTRACT MARKER**: 30 节点的 marker 注释未动 (与新代码一致),无需再同步更新。
