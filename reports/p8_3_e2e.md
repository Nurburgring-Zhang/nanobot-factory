测试在 `imdf/tests/integration/` 下, sys.path 不含 `backend/imdf/`, 找不到 `api.canvas_web`。

`imdf/api/canvas_web.py` 文件**实际存在** (有 `.bak` 备份, 1095 行), 但需要 `sys.path.insert(0, "backend/imdf")` 才能 import。

### 6.3 修复方案 (1 hour)

新建 `backend/imdf/tests/conftest.py`:
```python
import sys
from pathlib import Path
_BACKEND_IMDF = Path(__file__).resolve().parent.parent.parent
if str(_BACKEND_IMDF) not in sys.path:
    sys.path.insert(0, str(_BACKEND_IMDF))
```
