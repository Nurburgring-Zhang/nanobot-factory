## 3. 关键设计决策

### 3.1 SDKGenerator — 无外部依赖

不走 `openapi-generator-cli` (50 MB jar),直接基于 `paths` / `components.schemas` 字段字符串模板生成。理由:
- hermetic(测试无需 network / 无 java)
- 确定性输出(同输入同输出,便于缓存)
- `package_name` 强校验(`^[a-zA-Z][a-zA-Z0-9_-]{0,127}$`)
- `language` 不在白名单(`python/javascript/go`)→ `ValueError` → API 层映射 422

### 3.2 三套 SDK 模板输出结构

```
my-sdk.zip
├── MANIFEST.json          # 语言/文件清单
├── README.md              # 安装/使用示例
├── (python/)              # __init__.py + client.py + async_client.py + models.py + setup.py
├── (javascript/)          # index.js + types.js + package.json
└── (go/)                  # <pkg>.go + types.go + go.mod
```

每个模板都包含:
- **Python**: `IMDFClient` (sync, requests) + `AsyncIMDFClient` (async, httpx) + Pydantic `BaseModel` + 类型提示
- **JavaScript (ES6)**: `IMDFClient` class + `fetch` + JSDoc typedefs + `package.json` (type=module)
- **Go**: `Client` struct + `(c *Client) Request(...)` + struct 类型 + JSON tag + `go.mod`

### 3.3 SemanticSearchEngine — 双索引 hybrid

- **Vector side**: 自实现 `_VectorIndex`,TF-IDF + cosine similarity,支持 CJK bigram
- **BM25 side**: `_BM25Index`,优先用 `rank_bm25.BM25Okapi`,fallback 自实现 BM25 (k1=1.5, b=0.75)
- **Hybrid**: `score = alpha * vector_score + (1 - alpha) * bm25_score`,两边 min-max 归一到 [0,1]
- **持久化**: 可选 SQLite (`semantic_assets` + `semantic_index_log` 表),进程重启自动 rehydrate
- **CJK 处理**: `_tokenize` 检测 `\u4e00-\u9fff` 范围,额外 emit bigrams,所以中文无空格查询也能匹配

### 3.4 端点设计

| Method | Path | 行为 |
|---|---|---|
| GET | `/api/v1/sdk/languages` | 返回 SDKGenerator 支持的语言 + 元信息 |
| POST | `/api/v1/sdk/generate` | `language=javascript\|go` → application/zip (SDKGenerator);`python\|typescript\|both` → legacy JSON |
| POST | `/api/search/advanced` | hybrid search,接受 `{query, top_k, alpha}` → results list |
| POST | `/api/search/advanced/index` | `{asset_id, text, metadata}` → 索引 |
| GET | `/api/search/advanced/stats` | 返回 `{asset_count, size, corpus_size, bm25_count, vector_dim, ...}` |

## 5. 已知 trade-off / 后续 TODO

1. **Vector index 重建**: 每次 `add` 都 `_rebuild_index()` 全量重建 vocab/IDF,对 1k+ 资产性能 O(n²)。生产环境应在 `index_asset` 改为增量 IDF 更新 + 定期重建。
2. **CJK 跨语种融合**: 当 alpha=0.7 时,中文 query 在英文 corpus 上表现差(无 bigram overlap)。建议未来引入多语种向量模型。
3. **SDK zip 体积**: Python/Go SDK 单文件 ~5-15 KB,JS ~3-8 KB;可接受,无需压缩比调优。
4. **`/generate` 旧行为 vs 新行为**: 我用 `language in ("javascript", "go")` 作为分支条件,保留了 legacy 路径。后续可以迁移所有路径到 SDKGenerator。
