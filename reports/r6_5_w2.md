## 3. a11y — 无障碍能力清单

### 3.1 实现项 (符合 WCAG 2.1 AA)

| 能力 | 实现 | WCAG 条款 |
|---|---|---|
| **skip-link** | DOMContentLoaded 时自动注入 `<a class="a11y-skip-link" href="#a11y-main">`, Tab 首次聚焦时显示在左上角 | 2.4.1 Bypass Blocks |
| **focus-visible** | 全局 `*:focus-visible { outline: 3px solid #1f6feb; outline-offset: 2px }`, 鼠标点击不显示 | 2.4.7 Focus Visible |
| **v-label 装饰器** | 给元素加 aria-label (支持 i18n key 自动解析 + 切换语言时重新求值) | 4.1.2 Name, Role, Value |
| **tab order 报告** | 页面加载完 console 输出 7 列表格 (index/tag/id/text/ariaLabel), 存到 `window.__A11Y_TAB_ORDER__` | 2.4.3 Focus Order |
| **颜色对比度** | 6 组关键配色 (正文 / 弱化 / accent / danger / success / warning) 全部 ≥ 4.5:1 (实测 5.91 — 15.56) | 1.4.3 Contrast (Minimum) |
| **prefers-contrast** | 高对比度模式下加粗 outline 到 4px, 文字改纯黑 | 1.4.6 Contrast (Enhanced) |
| **prefers-reduced-motion** | 自动禁用动画和过渡 | 2.3.3 Animation from Interactions |
| **sr-only** | `.a11y-sr-only` 视觉隐藏但屏幕阅读器可读 | 通用 |

### 4.2 i18n API

```js
// 模板
<h1>{{ $t('nav.dashboard') }}</h1>
<p>{{ $t('error.network') }}</p>

// script
this.$t('common.confirm')                           // → '确认' / 'Confirm'
this.$t('test.greet', { name: 'Alice' })            // 参数插值

// 切换语言
this.$i18n.setLang('en-US')                         // → 触发 'i18n:lang-changed' 事件
this.$i18n.getLang()                                // → 'en-US'
this.$i18n.getSupported()                           // → ['zh-CN', 'en-US']

// 持久化
localStorage.getItem('i18n.lang')                   // → 'zh-CN' (启动时自动读)
```

### 4.3 Fallback 策略

- 当前语言 (zh-CN) 命中 → 返回该语言文案
- 当前语言缺失 → 自动 fallback 到 en-US
- 双语都缺失 → console.warn + 返回 key 字面量 (e.g. `'nav.dashboard'`)
