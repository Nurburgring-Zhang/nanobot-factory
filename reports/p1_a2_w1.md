## 关键设计

### 1. 哈希链审计
```python
chain_hash = SHA-256(prev_hash || canonical_json(payload))
```
- 每个审计记录存 `prev_hash` + `chain_hash` + `seq`
- `verify_audit_chain()` 重新计算并比对, 检测篡改
- 测试 35 验证了篡改检测: 修改 `payload` 后 `verify_audit_chain` 返回 `ok=False, broken_at_seq=<seq>`

### 2. PII 检测精度
- Chinese ID card: 18 位 + GB 11643-1999 checksum
- Credit card: 13-19 位 + Luhn checksum
- Bank card: 16-19 位 + Luhn + BIN 前缀
- Address: 收紧正则 (需省/市/区/县前缀, 防止 "身份证号" 误匹配)

### 3. 匿名化
- `ANON_USER_<8-hex>` 是 `SHA-256(user_id)[:8]` 的前 8 字符 — 不可逆
- PII 字段递归替换为 `[REDACTED]`, 非 PII 字段保留 (保留统计价值)
- 删除 consent records (因为同意是 subject 自愿, 匿名后无法追踪)

### 4. Portability 标准 Schema
```json
{
  "schema": "GDPR-Article20-v1",
  "schema_version": "1.0",
  "profile": {"name": "...", "email": "...", "phone": "...", "address": "..."},
  "data_categories": [{"category": "...", "value": {...}, "metadata": {...}, "created_at": "..."}],
  "consents": [{"purpose": "...", "status": "...", "version": "...", "recorded_at": "..."}]
}
```
