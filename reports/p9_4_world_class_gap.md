## 一、5 大顶级平台安全基线

### 1.1 Auth0 (Authentication)
- OAuth 2.1 + OIDC + SAML
- MFA / WebAuthn / Passkey
- Adaptive MFA (风险评分)
- Universal Login (托管登录页)
- 99.99% SLA + 零信任架构

### 1.2 HashiCorp Vault (Secrets)
- KV v2 + Dynamic Secrets (DB / AWS / GCP)
- Transit (加密即服务)
- 自动 90d 轮换
- PKI 证书签发
- Shamir Secret Sharing (Unseal)

### 1.3 Ory Keto / Casbin (Authorization)
- RBAC + ABAC + ReBAC 三合一
- DSL 策略 (`.rego` / `.conf`)
- 任何 DB 后端
- 分布式 + 高可用
- 亚毫秒级决策

### 1.4 DocuSign / 法大大 (E-Signature)
- 第三方 CA + 时间戳
- 实名认证 (KYC + 活体)
- Webhook + 审计
- 法律效力 (eIDAS / 电子签名法)

### 1.5 AWS KMS / Azure Key Vault (HSM)
- FIPS 140-2 Level 3 HSM
- 自动密钥轮换
- CloudTrail 审计
- BYOK (Bring Your Own Key)
- 国密 (中国区域)
