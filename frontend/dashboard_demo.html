♿ a11y 演示

本页已自动注入: skip-link (Tab 首次聚焦时显示在左上角) + focus-visible 全局焦点环 + reduced motion 媒体查询支持。
  • 按 Tab 键, 第一个聚焦元素应为左上角蓝色 "跳到主内容" 链接
  • 继续 Tab, 所有可聚焦元素显示 蓝色 3px outline
  • 打开 DevTools Console, 应看到 [a11y] Tab order (N focusable elements) + [a11y] Color contrast report
  • 使用屏幕阅读器 (NVDA / VoiceOver) 时, 按钮读屏由 v-label 装饰器注入 aria-label
// 打开 DevTools Console 查看 > window.__A11Y_TAB_ORDER__ // 完整 tab order 表 > window.__A11Y_CONTRAST__ // 颜色对比度报告 (全部应 passAA=true) > window.RbacCore.can('admin', 'create:user') // → true > window.RbacCore.can('viewer', 'create:user') // → false > localStorage.getItem('i18n.lang') // → "zh-CN" or "en-US"
