# IMDF UI架构设计 v3.0 — 直接呈现常用功能

对标: Scale AI / Labelbox / Supervisely / Figma

## 核心原则

1. **最常用的功能直接铺在首页** — 不需要点菜单
2. **左侧导航** — 快速切换业务模块
3. **快捷工具栏** — 悬浮在画布/列表上的操作入口
4. **全局拖拽上传** — 任何页面都可以拖文件进来

## 实现策略

### 第一阶段 — 替换当前坏掉的内联HTML (立即)
1. 把HTML_TEMPLATE从Python字符串中移到独立文件 `frontend/index.html`
2. 先实现**首页布局** (4指标卡 + 快捷操作区 + 最近任务) — 这是最直接的
3. 左侧导航固定 + 状态栏
4. 后端/static目录提供静态文件

### 第二阶段 — 核心业务页面
5. 数据集管理页 (表格+搜索+批量操作)
6. 标注工具页 (画布+AI预标注)
7. 工作流画布页 (迁移现有47节点)

### 第三阶段 — 完整功能
8. 团队管理/交付/审核/统计/设置
9. 用户认证集成
10. 国际化/主题
