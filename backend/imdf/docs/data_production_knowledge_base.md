# IMDF 多模态训练数据生产知识库 v1.0

## 一、AIGC文生图模型训练数据生产

### 1.1 生产方向与阶段

```
预训练阶段: 海量图文对(10亿+级别)
  - LAION-5B / DataComp / COYO-700M 风格
  - 爬虫→清洗→去重→NSFW过滤→质量评分→caption生成
  
微调阶段: 百万级高质量数据
  - 风格微调: 特定艺术家/画风的密集数据
  - 概念微调: DreamBooth风格(一个概念5-20张图)
  - LoRA: 低秩适配数据(50-200张高质量图)
  
RLHF/DPO阶段: 偏好数据
  - 同一prompt的不同输出→人工排序
  - 美学评分(Aesthetic Score)作为代理奖励
  
ControlNet/IP-Adapter阶段: 条件控制数据
  - 边缘图(Canny)→原图
  - 深度图(Depth)→原图  
  - 姿态图(OpenPose)→原图
  - 分割图(Segmentation)→原图
```

### 1.2 生产工具链

| 阶段 | 工具 | 用途 | GitHub Stars |
|------|------|------|-------------|
| 爬虫 | img2dataset | 大规模图片URL→本地下载 | 3.5K |
| 清洗 | clip-retrieval | CLIP去重/过滤 | 1.5K |
| caption | BLIP-2 / LLaVA | 自动生成图文描述 | 10K+ |
| caption | CogVLM2 / Qwen-VL | 中英文caption | 15K+ |
| 美学评分 | aesthetic-predictor | 图像美学质量评分 | 1K |
| NSFW检测 | CLIP-based NSFW | 不安全内容过滤 | 3K |
| 去重 | dhash / pHash | 感知哈希去重 | 内置 |
| 格式转换 | img2dataset | WebDataset/S3格式 | 3.5K |

### 1.3 质量标准

```
图片质量:
  - 分辨率: ≥512x512 (预训练), ≥1024x1024 (微调)
  - 美学评分: ≥5.5/10 (预训练), ≥7.0/10 (微调)
  - 去重: pHash距离 ≥20
  - NSFW: 分数 ≤0.5

Caption质量:
  - 准确性: 描述与图片内容一致
  - 丰富性: 包含主体、动作、场景、风格
  - 长度: 20-200 tokens
  - 语言: 英语为主,中文需特别标注
```

### 1.4 真实生产需求

1. **商业模型训练**: Midjourney v6/v7, DALL-E 4, Firefly 2 需要亿级高质量图文对
2. **风格迁移**: 特定画风(浮世绘/水墨/赛博朋克) 500-2000张
3. **LoRA定制**: 人物/产品/角色的50-200张多角度图
4. **ControlNet数据**: 边缘/深度/姿态/分割 各10万+对
5. **RLHF数据**: 10万+偏好对(需人工标注)
6. **产品图生成**: 电商产品白底图→场景图 5000+张

## 二、图片编辑模型训练数据生产

### 2.1 生产方向

```
Outpaint(扩图): 原图+扩展区域→完整图
  - 方向: 上下左右全方向
  - 扩展比例: 20%-200%
  - 一致性: 边缘过渡自然

Inpaint(局部重绘): 原图+蒙版→修改后图
  - 蒙版类型: 随机/规则/语义
  - 修改内容: 物体替换/背景更换/纹理修改
  - 质量: 局部与全局一致

超分辨率: 低清→高清
  - 倍数: 2x/4x/8x
  - 输入: 降采样/真实低清
  - 输出: 高质量细节

调色/滤镜: 原始→调色后
  - HSL调整/曲线/LUT
  - 风格迁移

去水印/去噪: 退化→清晰
  - 水印类型: 文字/Logo/半透明
  - 噪声类型: 高斯/椒盐/真实噪声
```

### 2.2 生产工具链

| 任务 | 开源工具 | 用途 |
|------|---------|------|
| 自动蒙版生成 | SAM / GroundingDINO | 语义分割生成蒙版 |
| Outpaint数据 | LaMa pipeline | 随机裁剪+填充→原图 |
| 超分数据 | Real-ESRGAN | 降采样→原图对 |
| 去噪数据 | noise2noise | 噪声→干净对 |
| 水印数据 | watermark-removal | 合成水印→原图 |

### 2.3 质量标准

```
Outpaint:
  - 边缘过渡PSNR ≥30dB
  - 无拼接痕迹
  - 扩展内容语义合理

Inpaint:
  - 修复区域与原图风格一致
  - 边缘无锯齿/色差
  - 语义正确(修车不能变修船)

超分:
  - 4x超分PSNR ≥28dB
  - 无伪影/振铃
  - 纹理细节真实
```

### 2.4 真实生产需求

1. **电商场景图**: 白底产品图→场景图(100万+对)
2. **人像修复**: 老照片修复/美颜(50万+对)
3. **AI扩图**: 社交媒体图片扩图(100万+)
4. **水印批量去除**: 版权水印训练数据(10万+张)
5. **背景替换**: 证件照/商品图背景替换(50万+)

## 三、视频模型训练数据生产

### 3.1 生产方向

```
文生视频: 文本→视频
  - 时长: 2-60秒
  - 帧率: 24-60fps
  - 分辨率: 720p-4K
  - 一致性: 时序平滑

图生视频: 图片+文本→视频
  - 起点: 静态图→动画
  - 运镜: 推拉摇移跟
  - 过渡: 淡入淡出/溶解/滑动

视频编辑: 视频+指令→编辑后视频
  - 局部修改: 换物体/换背景
  - 风格迁移: 视频→艺术风格
  - 时长编辑: 延长/缩短/插入

长视频: 多片段拼接
  - 镜头切换: 叙事连贯
  - 角色一致性: 跨镜头
  - 场景连续性: 时空一致
```

### 3.2 生产工具链

| 工具 | 用途 | Stars |
|------|------|-------|
| VideoCrafter2 | 文生视频 | 5K |
| AnimateDiff | 图生视频 | 15K+ |
| Stable Video Diffusion | 图生视频 | 8K |
| Wan Video | 高质量视频生成 | 新项目 |
| FFmpeg | 视频处理基础库 | 系统级 |

### 3.3 质量标准

```
单帧质量:
  - 分辨率 ≥720p
  - 无伪影/变形
  - 自然光照

时序质量:
  - 帧间变化平滑
  - 无闪烁/抖动
  - 运动合理

语义质量:
  - 主体一致性(不突变)
  - 背景连续性(场景不跳变)
  - 动作自然
```

### 3.4 真实生产需求

1. **社交媒体视频**: 抖音/视频号 9:16 15-60秒(100万+)
2. **产品演示**: 产品动态展示(10万+)
3. **教学视频**: 步骤分解动画(5万+)
4. **广告视频**: 品牌宣传15-30秒(1万+)
5. **视频修复**: 老视频超分/上色(10万+)

## 四、影视剧训练数据生产

### 4.1 生产方向

```
多镜头叙事: 剧本→多镜头视频
  - 镜头规划: 分镜脚本→镜头列表
  - 角色一致性: 跨镜头脸不崩塌
  - 场景连续性: 空间/光照/道具一致

故事板生成: 文字→分镜图
  - 镜头类型: 远景/中景/特写/过肩
  - 运镜标注: 推拉摇移跟升降
  - 情绪标注: 每个镜头的情绪目标值

长视频一致性: 跨片段>10分钟
  - 角色记忆: 穿衣/状态/位置
  - 道具追踪: 指定物体在场景中
  - 时间连续性: 白天/黑夜/天气
```

### 4.2 生产工具链

| 工具 | 用途 |
|------|------|
| Toonflow (9.8K★) | 一站式AI短剧(剧本→分镜→角色→视频) |
| Jellyfish (3.8K★) | shot-state-machine 架构 |
| ArcReel (2.5K★) | 角色设计图+线索追踪 |
| GPT-Image2+Seedance | 灰白稿故事版→视频 |

### 4.3 质量标准

```
镜头级:
  - 构图: 专业影视构图(三分法/引导线)
  - 运镜: 不违反180度法则
  - 剪辑: 匹配剪辑/Murch's Rule

角色级:
  - 一致性: 跨镜头相似度≥95%
  - 表情: 自然/不恐怖谷
  - 动作: 符合人体工学

叙事级:
  - 节奏: 起承转合结构完整
  - 情绪: 有情感曲线
  - 逻辑: 时空连续不跳戏
```

### 4.4 真实生产需求

1. **短剧生产**: 3-5分钟竖屏短剧(月产100+部)
2. **动画剧集**: 10集x10分钟动画剧集
3. **游戏CG**: 30-60秒高质量游戏宣传片
4. **影视预演**: 电影Previs(预览)动画
5. **广告剧情**: 30秒叙事广告

## 五、绘本训练数据生产

### 5.1 生产方向

```
多页故事: 文字→多页绘本
  - 主角一致性: 跨页角色形象统一
  - 风格一致性: 全书画风统一
  - 叙事连贯: 页间动作连续
  - 文字布局: 图文搭配合理

绘本风格: 
  - 水彩/油画/拼贴/剪纸/数字绘画
  - 儿童画/漫画/写实/抽象
  - 东西方风格

文字处理:
  - 中英文排版
  - 手写体/印刷体
  - 文字与画面融合
```

### 5.2 生产工具链

| 工具 | 用途 |
|------|------|
| GPT-Image2 | 高质量绘本单页生成 |
| ComfyUI + SDXL | 风格化绘本生成 |
| IP-Adapter | 角色一致性保持 |
| StoryDiffusion | 多页故事生成 |

### 5.3 质量标准

```
每页:
  - 分辨率: ≥1024x1024
  - 角色: 与前一页一致
  - 文字: 清晰可读/无错字
  - 构图: 适合儿童阅读

全书:
  - 画风: 全册统一(水彩/油画等不可混)
  - 角色: 从头到尾一致(肤色/服装/特征)
  - 色调: 全书风格统一或渐变
  - 文字量: 适合目标年龄段

年龄段适配:
  - 0-3岁: 大块颜色/简单形状/少量文字
  - 3-6岁: 简单故事/每页1-3句话
  - 6-9岁: 复杂故事/每页3-5句话
  - 9-12岁: 章节书/图文并重
```

### 5.4 真实生产需求

1. **儿童绘本**: 32页标准绘本(月产10-50本)
2. **教育绘本**: 科普/识字/数学启蒙绘本
3. **定制绘本**: 以特定孩子为主角的个性化绘本
4. **多语言绘本**: 同绘本不同语言版本
5. **互动绘本**: 翻翻书/立体书数字化版本
