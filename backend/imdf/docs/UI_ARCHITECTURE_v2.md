## 三、交互规范

1. **左侧菜单固定** — 永远可见，当前菜单高亮
2. **面包屑导航** — 显示当前位置 (如: 数据生产 > 数据集管理 > 版本管理)
3. **右键操作** — 表格行右键弹出操作菜单
4. **批量操作** — 支持Shift多选 + 顶部批量操作按钮
5. **快捷键** — Ctrl+S保存, Ctrl+Z撤销, Delete删除
6. **搜索入口** — 全局搜索框 (搜数据集/人员/任务)
7. **通知中心** — 右上角铃铛图标, 显示待办事项
8. **拖拽上传** — 数据生产页面支持拖拽文件上传

## 五、实现策略

1. **前端独立** — 不再用HTML_TEMPLATE内联, 改用`frontend/index.html` + JS/CSS独立文件
2. **后端只提供API** — FastAPI只返回JSON, 前端用fetch调用
3. **渐进式实现** — 先做核心框架(导航+菜单+路由), 再逐个填充面板
4. **可扩展** — 一级菜单、二级菜单、功能面板全部用配置驱动, 不硬编码
