# 🎯 General AIGC Enhanced 使用说明

## 📦 部署包内容

### 核心文件
- `start.bat` - Windows一键启动脚本
- `start.sh` - Linux/macOS启动脚本  
- `enhanced_startup.py` - 增强启动脚本
- `test_all.bat` - 完整功能测试
- `README.md` - 详细文档
- `QUICK_START.md` - 快速指南

### 验证工具
- `功能验证脚本.py` - 全面功能验证
- `部署修复脚本.py` - 自动问题修复
- `check_system.bat` - 系统环境检查

### 应用结构
- `frontend/` - React前端构建文件
- `backend/` - Python后端和AI模块
- `config/` - 配置文件

## 🚀 快速启动

### Windows用户
1. **解压部署包** 到任意目录
2. **双击 `start.bat`** 启动应用
3. **浏览器访问** http://localhost:3000

### Linux/macOS用户
1. **解压部署包**
2. **运行** `./start.sh`
3. **浏览器访问** http://localhost:3000

### 增强启动（推荐）
```bash
python enhanced_startup.py
```

## 🛠️ 故障排除

### 常见问题解决

**1. 虚拟环境路径错误**
```
运行: python 部署修复脚本.py
```

**2. 后端模块导入失败**
```
运行: python enhanced_startup.py
```

**3. ComfyUI/WebUI集成问题**
```
检查: ComfyUI和WebUI目录是否存在
运行: python 功能验证脚本.py
```

**4. 端口占用**
```
停止占用端口的进程
或修改配置文件中的端口号
```

### 手动修复
```bash
# 1. 修复所有问题
python 部署修复脚本.py

# 2. 验证修复结果  
python 功能验证脚本.py

# 3. 重新启动
python enhanced_startup.py
```

## 🔧 开发说明

### 本地开发
```bash
# 1. 激活虚拟环境
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate     # Windows

# 2. 安装依赖
pip install -r backend/requirements.txt

# 3. 启动开发服务器
python enhanced_startup.py
```
